# barnkanalen-spel
A backup of the Swedish childrens televisions Flashgames
Some games only work if hosted on a web server. A website may be built. Until then use: https://chrome.google.com/webstore/detail/web-server-for-chrome/ofhbbkphhbklhfoeikjpcbhemlocgigb

Nallebeorn provided most of the files in the end and we changed the file structure to fit this project. This may result in some bugs, missing files or not being able to run offline. His unchanged file structure is available here: ([zip](https://archive.org/download/spel_20201125/nallebjoern.zip),[7z](https://archive.org/download/spel_20201125/nallebjoern.7z)) if you experince bugs. He has also added them to BlueMaxima's [Flashpoint](https://bluemaxima.org/flashpoint/)
